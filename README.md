🌿 ElderCARE Harmony NGO

A full-stack web application designed to support elderly care services by connecting users with emergency help, caregiving support, and NGO services.

---

## 🚀 Live Demo

* 🌐 Frontend: https://elder-care-harmony-ngo.vercel.app
* ⚙️ Backend: https://your-backend.onrender.com

---

## 📌 Features

* 🏠 Informative home page
* 📞 Contact form (stores data in MongoDB)
* 🚨 Emergency assistance section
* 👵 Elder care services
* 🌐 Fully deployed (Vercel + Render)
* 🔗 API integration with backend

---

## 🛠️ Tech Stack

**Frontend**

* React (Vite)
* Tailwind CSS

**Backend**

* Node.js
* Express.js

**Database**

* MongoDB Atlas

**Deployment**

* Vercel (Frontend)
* Render (Backend)

---

## 📂 Project Structure

```
ElderCARE-Harmony-NGO-
│
├── frontend/   # React frontend
├── backend/    # Node.js backend
└── README.md
```

---

## ⚙️ Setup Instructions

### 1️⃣ Clone Repository

```
git clone https://github.com/apurva-ak47/ElderCARE-Harmony-NGO-.git
cd ElderCARE-Harmony-NGO-
```

---

### 2️⃣ Backend Setup

```
cd backend
npm install
```

Create `.env` file:

```
MONGODB_URI=your_mongodb_connection_string
PORT=5000
JWT_SECRET=your_secret_key
```

Run backend:

```
npm start
```

---

### 3️⃣ Frontend Setup

```
cd frontend
npm install
npm run dev
```

---

## 🌍 Environment Variables

### Backend (.env)

* `MONGODB_URI` → MongoDB connection string
* `PORT` → Server port
* `JWT_SECRET` → Secret key

### Frontend (Vercel)

* `VITE_API_URL` → Backend deployed URL
